Create a Slideshow:

If you have a set of related images that you would like to display as a slideshow, you may select them using the image assist function.

  1. Click the image assist icon under the text box;
  2. Select the image you would like to insert. Fill in Title and Caption if you would like to add information.
  3. Click "Insert"
  4. You will see something like this:
    [img_assist|nid=22|title=Image One|desc=Image Description|link=node|align=left|width=100|height=75]
    This is the image filter which will automatically insert the image.
  5. To include more images in the slideshow, repeat steps 1 thru 4. You will see a list something like this:
    [img_assist|nid=22|title=Image One|desc=Image Description|link=node|align=left|width=100|height=75]
    [img_assist|nid=23|title=Image Two|desc=Image Description|link=node|align=left|width=100|height=75]
    [img_assist|nid=24|title=Image Three|desc=Image Description|link=node|align=left|width=100|height=75]
    This will display three images in a row. When any image is clicked, a slideshow will automatically begin.
  6. If you would like only ONE image visible even though the slideshow contains multiple images. To do this, you must add the following to the "align=" element. It should look like this:
    [img_assist|nid=22|title=Image Title|desc=Image Description|link=node|align=left lightbox_hide_image|width=100|height=75]
  7. Do this to ALL IMAGES that you want included in the slideshow, but not visible on the page. The image without this tag will serve as the trigger. So, a complete slideshow will look like this:
    [img_assist|nid=22|title=Image One|desc=Image Description|link=node|align=left|width=100|height=75]
    [img_assist|nid=23|title=Image Two|desc=Image Description|link=node|align=left ightbox_hide_image|width=100|height=75]
    [img_assist|nid=24|title=Image Three|desc=Image Description|link=node|align=left ightbox_hide_image|width=100|height=75]
